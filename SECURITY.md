# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

Para reportar uma vulnerabilidade de segurança, **não abra uma issue pública**.

Envie um e-mail para: **mauricio@regroup.com.br**

Inclua:
- Descrição detalhada da vulnerabilidade
- Passos para reproduzir
- Impacto potencial
- Sugestão de correção (opcional)

Responderemos em até **48 horas** com a confirmação de recebimento.

## Disclosure Policy

Após confirmação e correção da vulnerabilidade:
- Notificaremos o reporter
- Publicaremos um advisory no GitHub Security Advisories
- Crédito será dado ao reporter (se desejado)
